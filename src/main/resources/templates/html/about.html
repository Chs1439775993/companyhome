.NET开发工程师

> 职位描述

  1. 理解业务需求,并参与业务流程与部分功能界面的设计；
  2. 遵守开发规范的要求,主动进行单元测试,保障开发质量；
  3. 完成模块文档、示例的编写,提高模块的性能及通用性,主动分享技术及工作经验；
  4. 独立完成后段模块或接口的开发工作,如编写符合RESTful风格的API、windows值守服务、数据采集程序。

> 任职要求

  1. 熟悉掌握C#语言及.NET框架平台机制,了解3.5以上常用新特性；
  2. 熟悉三层架构,ORM、MVC等主流框架,了解MVC内部机制,理解HTTP协议及RESTful架构风格；
  3. 熟悉SQL Server等关系型数据库,掌握增删改查、存储过程、触发器等基本操作；
  4. 熟练操作VS、SVN等开发工具;
  5. 了解HTML、CSS、JavaScript等基础前端知识。

JAVA 工程师

> 职位描述

  1. 负责APP和Web项目和后端各服务组件的技术需求分析、功能开发、业务维护；
  2. 根据需求反馈,新增或优化工程组件,并协助测试工程师完成产品测试相关工作；
  3. 负责对以往业务代码的梳理、整合及优化,并及时更新相关技术文档；
  4. 负责Java开发相关的技术文档、技术手册的编写和维护。

> 任职要求

  1. 计算机或相关专业；
  2. 熟悉Java开发语言,使用过常用的J2EE开发框架,如SpringBoot、Spring MVC、Mybatis等；
  3. 熟悉常用的软件设计模式、数据结构,具备扎实的编程基本功和良好的编程习惯；
  4. 熟悉常用的关系数据库、NoSQL数据库,如MySQL、HBase、MongoDB；
  5. 熟悉常用的数据缓存技术,如Redis、Memcache等；
  6. 了解前端开发相关技术,如HTML/CSS/JS等。
